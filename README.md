# ai-realestate-website

Sitio estático de generación de leads para agente inmobiliaria en Asunción, Paraguay.

## Stack
- Astro 4
- Tailwind CSS
- TypeScript
- Formspree
- Vercel

## Variables públicas
Ver `.env.example`.

## Desarrollo
```bash
npm install
npm run dev
npm run build
npm run preview
```
